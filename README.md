## General info
This is a Markdown previewer app.
	
## Technologies
Project is created with:
* JavaScript
* SCSS
* react: 18.2.0
* redux: 4.2.0
* react-redux: 8.0.2
* reduxjs/toolkit: 1.8.5
* react-markdown: 8.0.3
* react-syntax-highlighter: 15.5.0

	
## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
